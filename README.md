# 赛博玄数 Cyber-Mantic

> 时空齿轮的数字化探索

---

## 缘起

两千年前，中国先祖们用干支构建了一套多周期嵌套的时间编码系统——十天干、十二地支、六十甲子，年月日时层层叠加，描绘出时间的纹理与节奏。而我，2016年，坐在教室里听着老师用德语讲傅里叶变换将周期函数分解为正弦波的叠加……

这里是不是有些理念有重叠？

用有限的符号，去捕捉时空运转的内在规律。

**赛博玄数**的初心，是用现代技术重新审视这套古老的符号系统：

- 不是要证明它"灵"或"不灵"；
- 而是想看看，当我们剥离神秘主义的外衣，用代码精确复现每一步推演，用多理论交叉验证每一个结论，这套系统会呈现出怎样的面貌；
- 想让更多人有机会理解，中国先祖们是如何观察这片星空、思考这个世界、编码这条时间之河的；

术数是中华文明对宇宙规律的一种探索记录，值得被更多人看见、理解、思考。

## 这是什么

**赛博玄数**融合了八种传统术数理论，用代码实现排盘计算，用AI辅助意义解读：

| 术数 | 核心思想 | 信息输入 |
|------|----------|----------|
| **八字** | 出生时刻的时空编码，蕴含生命的原始程序 | 出生年月日时 |
| **紫微斗数** | 星曜在十二宫的分布，描绘人生各领域的格局 | 出生年月日时 |
| **奇门遁甲** | 九宫八门九星八神的多层叠合，捕捉时空能量场 | 起局时间 |
| **大六壬** | 四课三传十二天将，叙述事态的起承转合 | 起课时间 |
| **六爻** | 铜钱落定的刹那，问题与答案的量子纠缠 | 摇卦数字 |
| **梅花易数** | 心念与外应的同步性，随机中的必然 | 任意数字或文字 |
| **小六壬** | 六神煞的快速轮转，日常决策的简便参考 | 时间 |
| **测字** | 汉字的形音义拆解，文字中的全息映射 | 一个汉字 |

### 设计理念

**计算与解读分离**：所有干支推演、星曜排布、卦象生成都由代码精确实现，结果可追溯、可验证。AI只负责将计算结果转译为人话，不参与核心推演。这样你看到的每一个天干地支，都有明确的算法来源，而非黑箱输出。

**多理论交叉验证**：单一理论可能有偏差，但当八字说"今年利于变动"，奇门说"此时宜动不宜静"，六爻也呈现出"动而有利"的卦象时，这个信号就更值得关注。系统会自动检测理论间的一致与分歧，让你看到完整的图景。

**冲突透明化**：当不同理论给出矛盾结论时，系统不会强行调和或隐藏，而是如实呈现分歧，说明各自的判断依据，让你自己权衡。

## 它能做什么

1. **时空状态分析**：输入一个时间点，系统会用多种理论解析这一刻的"时间质地"——它处于什么周期、携带什么属性、与其他时间点有怎样的关系

2. **个体命盘解读**：输入出生信息，系统会排出八字、紫微等命盘，分析五行分布、格局特点、大运走势

3. **具体问题占断**：针对特定问题起卦，系统会生成卦象并解析其中的信息

4. **AI对话深入**：对分析结果有疑问，可以继续追问，AI会结合你的具体情况展开讨论

---

## 它不能做什么

术数是古人观察天地规律的经验总结，不是预知未来的水晶球：

- 不能预测随机事件（彩票号码、明天股价）
- 不能代替专业判断（医疗、法律、投资请咨询专业人士）
- 不能保证绝对准确（任何模型都有其有效边界）

正如托勒密的地心说能够准确预测行星位置却基于错误的宇宙模型，术数可能在某些领域呈现出统计上的规律性，但其背后的机制可能与古人的解释完全不同。保持好奇，也保持审慎。

## 为什么开源

术数知识在历史上常被少数人垄断，要么秘而不宣，要么故作高深。

开源意味着：
- 每一步计算都可以被审视和验证
- 每一个理论都可以被讨论和改进
- 每一个人都可以参与这场跨越时空的对话

我们相信，真正有价值的知识不怕被看见。先祖们观察星空、推演历法、编纂术数，不是为了让后人在黑暗中摸索，而是为了点亮理解世界的一盏灯。
