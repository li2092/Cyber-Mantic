## 🎯 核心设计理念

### 产品定位
**赛博玄数是文化洞察与个人反思工具，非占卜或心理咨询服务。**

### 设计原则
1. **去神秘化**：用现代语言解释传统概念
2. **关怀优先**：预警机制是保护而非惩罚
3. **隐私至上**：数据本地存储，敏感检测本地完成
4. **渐进体验**：对话式交互降低使用门槛
