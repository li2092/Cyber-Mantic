## 2. 文档内容规范

### 2.1 写作风格要求

每篇文档遵循以下原则：

- **去神秘化**：用现代语言解释传统概念
- **科学态度**：明确说明这是文化洞察工具，非科学预测
- **心理关联**：适当关联现代心理学概念
- **伦理边界**：强调仅供参考，重大决策应咨询专业人士
- **连续叙述**：使用流畅的段落叙述，避免大量列表堆砌

### 2.2 敏感术语中性化

| 传统术语 | 中性化表述 |
|----------|------------|
| 官鬼 | 责任与挑战因素 |
| 白虎 | 行动力指标 |
| 凶神 | 需关注因素 |
| 吉神 | 积极因素 |
| 孤辰寡宿 | 独立性特征 |
| 桃花 | 社交吸引力 |

### 2.3 每篇文档结构

1. **引言**：一句话概括主题
2. **核心概念**：关键术语及现代解读
3. **详细内容**：主体知识讲解
4. **现代视角**：与心理学/系统论的关联
5. **在赛博玄数中的应用**：后台分析角色说明
6. **注意事项与边界**：局限性与伦理提醒
7. **求助资源**：专业帮助联系方式
