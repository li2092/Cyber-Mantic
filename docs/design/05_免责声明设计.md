## 1. 触发场景

| 场景 | 触发内容 | 频率 |
|------|---------|------|
| 首次启动 | 完整协议 + 语言选择 | 仅一次 |
| 切换语言 | 对应语言的完整协议 | 每次切换 |
| 问道发送 | 简版提示 | 每次会话首次 |
| 推演分析 | 简版提示 | 每次会话首次 |
| 未成年八字 | 未成年人提示 | 检测到时 |

## 6. 语言切换触发

用户在界面右上角切换语言时：

1. 切换界面语言
2. 弹出对应语言的完整免责声明
3. 用户需重新勾选确认
4. 刷新问道模块欢迎词
