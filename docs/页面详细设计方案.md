## 一、问道页面（WendaoPage）

### 1.3 组件清单

| 组件 | 功能 | 数据来源 |
|------|------|---------|
| **StageIndicator** | 显示当前对话阶段(1-5) | ConversationService.stage |
| **ChatBubble** | AI/用户消息气泡 | 对话历史 |
| **ProgressCard** | 对话进度百分比 | ConversationService.progress |
| **BaziMiniCard** | 简化八字显示(4柱) | context.bazi_result |
| **XiaoLiuRenCard** | 小六壬速断结果 | context.xiaoliu_result |
| **QuickInputGrid** | 快捷操作2x2网格 | 固定选项 |
| **NumberPad** | 1-9数字快速输入 | 用户输入 |

### 1.4 交互说明

| 触发 | 动作 |
|------|------|
| 发送消息 | → AI回复 + 更新进度 + 可能更新八字卡片 |
| 点击快捷按钮 | → 自动填入对应指令到输入框 |
| 阶段完成 | → 阶段指示器高亮下一阶段 |
| 对话完成(阶段5) | → 显示[保存对话]按钮 |

### 2.3 结果区4个Tab详情

#### Tab 1: 摘要
- 综合判断（吉/中吉/平/凶）+ 星级
- 置信度百分比
- 核心结论（3-5句话）
- 关键建议列表

#### Tab 2: 详细分析
- Markdown格式长文本
- 支持折叠/展开各节
- 术语可点击查看解释

#### Tab 3: 理论结果
- 每个理论一张卡片
- 卡片内容：理论名、判断、置信度、关键点
- 冲突提示（如有）

#### Tab 4: 可视化
- 八字四柱图
- 五行分布饼图/条形图
- 紫微星盘图（如选择紫微）
- 大运流年时间线

| 组件 | 功能 |
|------|------|
| **InputPanel** | 出生信息、问题、辅助信息输入 |
| **TheorySelector** | 理论多选复选框 |
| **ProgressPanel** | 进度条+日志滚动区 |
| **ResultTabs** | 4个Tab切换 |

| **TheoryResultCard** | 单个理论结果卡片 |
| **BaziChart** | 八字可视化 |
| **WuxingPieChart** | 五行饼图 |
