## 六大核心功能

### 💬 问道 —— 像聊天一样完成分析

**最具创新的对话式体验**

不用填表格，不用学术语。就像和一位懂你的智者对话：

```
你：最近工作压力很大，在考虑要不要换个环境

AI：我理解。能告诉我你的出生日期吗？我来帮你看看现在的运势走向...

你：1990年5月15日

AI：好的。你目前正处于一个转换期，从八字来看...
    你还记得2022年有没有发生什么工作上的变动？

你：对！那年正好升职了

AI：这验证了分析的准确性。接下来我来详细说说未来一年的趋势...
```

**5阶段渐进深入**：破冰 → 信息收集 → 深度分析 → 回溯验证 → 完整报告

**技术亮点**：
- Kimi自然语言解析，智能理解用户输入
- 三级时辰分类（确定/不确定/未知）
- 多理论融合分析引擎
- 持续问答，报告后可继续追问

---

### ⚡ 推演 —— 一键直达报告

**适合目标明确的用户**

知道自己想问什么？直接输入所有信息，系统自动选择最适合的理论组合，一次生成完整报告。

- 多理论融合分析（3-5个理论交叉验证）
- 智能理论选择，自动匹配最适合的理论组合
- 8理论并行计算，实时进度显示
- 智能冲突检测与仲裁
- 3-5分钟出结果
