## 一、首次启动

### 1.1 启动方式

双击桌面图标或运行程序后，首次启动会显示：
1. **免责声明对话框**：阅读并同意使用协议
2. **引导对话框**：快速了解主要功能

### 1.2 配置AI服务（必须）

首次使用需要配置至少一个AI服务密钥：

1. 点击顶部【设置】标签
2. 在"AI API 配置"区域，输入您的API密钥
3. 点击【保存配置】

**支持的AI服务**：
- **Claude**（推荐，分析质量最佳）
- **Deepseek**（国内访问稳定，速度快）
- **Gemini**（Google服务）
- **Kimi**（国内服务，用于自然语言解析）

> 💡 **提示**：建议配置多个API作为备用，系统会智能切换和故障转移

### 1.3 选择主题（可选）

在设置页面可以切换界面主题：
- **清雅白**：适合白天使用
- **墨夜黑**：适合夜间使用，护眼
- **禅意灰**：柔和中性，长时间阅读

## 三、问道：对话式分析

问道是赛博玄数的主打功能，像和一位智者对话，轻松完成深度分析。

### 3.1 基本流程

```
说出问题 → AI引导补充信息 → 回溯验证 → 生成报告 → 持续追问
```

### 3.2 五阶段对话

| 阶段 | 您需要做的 | AI会做的 |
|------|-----------|---------|
| **破冰** | 描述您的问题 | 初步判断，引导后续 |
| **信息收集** | 提供出生日期、性别等 | 展示可用的分析理论 |
| **深度补充** | 补充时辰、随机数等 | 启动多理论分析 |
| **结果验证** | 确认过去事件是否准确 | 校验分析方向 |
| **完整报告** | 阅读报告，继续追问 | 生成详细分析，持续答疑 |

### 3.3 对话示例

```
您：最近工作压力很大，在考虑要不要换个环境

AI：我理解。能告诉我你的出生日期吗？我来帮你看看现在的运势走向...

您：1990年5月15日

AI：好的。你目前正处于一个转换期，从八字来看...
    你还记得2022年有没有发生什么工作上的变动？

您：对！那年正好升职了

AI：这验证了分析的准确性。接下来我来详细说说未来一年的趋势...
```

### 3.4 使用技巧

- ✅ 问题描述越具体，分析越有针对性
- ✅ 可以随时请AI解释不懂的术语
- ✅ 报告生成后可以继续追问细节
- ✅ 回溯验证有助于确认分析方向
- ❌ 不要一次问多个不相关的问题

## 四、推演：快速生成报告

推演功能适合已经明确知道自己想问什么的用户，一次输入所有信息，直接生成完整报告。

### 4.1 基本流程

```
填写信息 → 选择理论 → 开始分析 → 查看报告
```

### 4.2 填写分析信息

#### 必填信息

| 字段 | 说明 | 示例 |
|------|------|------|
| **问题类别** | 选择您想咨询的方向 | 事业发展、感情婚姻、财运投资等 |
| **问题描述** | 详细描述您的问题 | "最近在考虑是否跳槽，想了解未来半年的事业运势" |
| **出生日期** | 您的出生年月日 | 1990年5月15日 |
| **出生时辰** | 您的出生时间段 | 上午9-11点（巳时） |
| **性别** | 用于命理计算 | 男/女 |

#### 可选信息（提高准确度）

| 字段 | 说明 | 适用理论 |
|------|------|---------|
| **出生地点** | 用于真太阳时校正 | 八字、紫微 |
| **随机数字** | 心中想一个数字 | 梅花易数 |
| **六爻数字** | 6个1-3的数字 | 六爻 |
| **测字汉字** | 心中想一个字 | 测字 |
| **当前方位** | 您面朝的方向 | 奇门遁甲 |

### 4.3 选择分析理论

系统支持两种模式：

**自动推荐（默认）**
- 系统根据您的问题和信息完备度自动选择3-5个最适合的理论
- 多理论交叉验证，结果更可靠

**手动选择**
- 勾选您想使用的具体理论
- 适合对术数有一定了解的用户

**八大理论简介**：

| 理论 | 特点 | 适合问题 |
|------|------|---------|
| **八字** | 分析人生整体格局 | 性格、运势、大方向 |
| **紫微斗数** | 十二宫位详细分析 | 各领域运势细节 |
| **奇门遁甲** | 时空能量分析 | 具体事件、时机选择 |
| **大六壬** | 事态发展预测 | 复杂局面分析 |
| **六爻** | 专事专断 | 具体问题的是非判断 |
| **梅花易数** | 快速起卦 | 日常小事、即时决策 |
| **小六壬** | 最简快速 | 出行、简单吉凶 |
| **测字** | 文字解读 | 趣味性分析 |

### 4.4 开始分析

1. 确认信息填写完整
2. 点击【开始分析】按钮
3. 观察实时进度显示（通常1-3分钟）

> ⏳ 分析过程中请勿关闭程序

### 4.5 查看分析报告

报告包含以下部分：

| 部分 | 内容 |
|------|------|
| **执行摘要** | 200-300字的核心结论 |
| **各理论分析** | 每个理论的详细解读 |
| **理论共识度** | 多理论结论的一致性分析 |
| **冲突说明** | 不同理论观点差异的解释 |
| **行动建议** | 具体可执行的建议 |
| **注意事项** | 需要关注的风险点 |

### 4.6 报告交互

- **术语解释**：点击专业术语，侧边栏显示解释
- **继续追问**：基于报告内容与AI对话
- **报告对比**：选择多份报告进行AI对比分析

### 4.7 导出报告

支持三种导出格式：
- **PDF**：适合打印或正式存档
- **Markdown**：适合编辑或发布
- **JSON**：适合数据备份

## 九、常见问题

### Q1：需要懂术数才能使用吗？

**不需要**。系统会自动选择合适的理论，AI会用通俗易懂的语言解读结果。您只需要提供基本信息和问题即可。

### Q2：分析结果准确吗？

赛博玄数通过多理论交叉验证提高结果的参考价值。但请注意：
- 分析结果仅供参考，不构成专业建议
- 重大决策请结合实际情况综合判断
- 命运掌握在自己手中，工具只是辅助

### Q3：为什么需要出生时间？

传统术数以出生时间作为分析基础，认为出生时刻的时空信息与人的特质有关联。出生时间越精确，分析越有针对性。

### Q4：如果不知道确切出生时间怎么办？

- 可以询问父母或查看出生证明
- 如果实在不知道，可以选择"时辰不详"
- 系统会根据可用信息进行分析，但准确度会降低
- 问道模式支持通过回溯验证推断时辰

### Q5：分析需要多长时间？

通常1-3分钟。时间取决于：
- 选择的理论数量
- AI服务的响应速度
- 网络状况

### Q6：我的数据安全吗？

- 分析数据存储在您的本地设备（SQLite数据库）
- 仅在分析时将必要信息发送给AI服务
- 您可以随时删除所有历史记录
- 无需注册账号，不收集个人身份信息

### Q7：可以给别人分析吗？

可以。在填写信息时输入对方的出生信息即可。但请确保：
- 获得对方的知情同意
- 尊重他人隐私

### Q8：为什么不同理论的结论有时不一致？

这是正常现象。不同理论从不同角度分析问题：
- 系统会显示理论共识度
- 当多个理论结论一致时，参考价值更高
- 冲突部分会有详细说明
- 可以在AI对话中请AI解释差异原因

### Q9：问道和推演有什么区别？

| 对比项 | 问道 | 推演 |
|-------|------|------|
| 交互方式 | 对话式，渐进引导 | 表单式，一次输入 |
| 适合用户 | 新用户，不确定问什么 | 老用户，目标明确 |
| 信息收集 | AI智能追问 | 一次性填写 |
| 回溯验证 | 有 | 无 |
| 追问方式 | 对话中直接追问 | 报告生成后追问 |

### Q10：典籍内容是谁写的？

典籍内容由项目组撰写，遵循以下原则：
- 用现代语言解释传统概念
- 去神秘化，科学理性态度
- 关联现代心理学概念
- 明确局限性和使用边界
