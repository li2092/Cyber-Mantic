基于用户的使用数据，生成画像分析报告。此任务使用 DeepSeek-Reasoner 深度思考模型。

## 分析要求

请使用深度推理能力，从以下维度分析用户画像：

### 1. 使用模式分析
- 使用频率趋势
- 时间规律
- 功能偏好

### 2. 关注重心分析
- 主要关注领域
- 关注点变化趋势
- 潜在需求推测

### 3. 决策风格分析
- 谨慎型/果断型/犹豫型
- 信息需求程度
- 对分析结果的接受模式

### 4. 个性化建议
- 推荐探索的理论/功能
- 使用方式优化建议
- 心理健康相关提示（如有必要）

## 输出格式

使用亲切的第二人称，像是一位了解用户的老朋友在给予建议。

字数控制在400-600字。

## 注意事项

- 分析基于数据，避免过度推测
- 语气温和正面
- 如发现异常模式（如过度使用），委婉提醒
