- 标题：{{document_title}}
- 来源：{{document_source}}

## 文档内容

{{document_content}}

## 用户笔记（如有）

{{user_notes}}

## 心得生成要求

### 1. 读后感悟
生成一段个人化的读后感悟，仿佛是用户自己写的学习心得。

### 2. 重点回顾
回顾文档中的重点内容。

### 3. 个人理解
用通俗的语言表达对内容的理解。

### 4. 实践计划
如何将所学应用到实践中。

### 5. 疑问记录
可能存在的疑问和后续学习方向。

## 输出格式

使用第一人称叙述，像是一篇个人学习日记。
总字数控制在400-600字。

## 风格要求

- 真实自然
- 有个人思考
- 态度谦虚好学
