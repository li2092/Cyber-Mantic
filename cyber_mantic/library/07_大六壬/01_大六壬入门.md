# 大六壬入门

> 大六壬与奇门遁甲、太乙神数并称"三式"，是中国古代最高深的预测术之一——以天地人三才为框架，演绎事物发展的完整脉络。

---

## 一、什么是大六壬

大六壬，简称"六壬"，是中国传统术数中最古老、最精密的预测体系之一。它与奇门遁甲、太乙神数并称为"三式"，被誉为术数之王。

"六壬"之名，源于天干中的"壬"字。十天干中，壬癸属水，壬为阳水。水为万物之源，主流动变化，最能体现事物发展的动态过程。而"六"字，则与六十甲子中的六壬日（壬子、壬寅、壬辰、壬午、壬申、壬戌）有关。

大六壬的核心结构是"四课三传"：

- **四课**：代表事物的四个基本方面，是分析的静态框架
- **三传**：代表事物发展的三个阶段，是分析的动态过程

通过四课三传，大六壬能够描绘事物的起因、经过、结果，以及其中各种力量的相互作用。这种结构化的分析方式，使大六壬成为传统术数中逻辑性最强、最具系统性的方法。

从现代视角来看，大六壬可以被理解为一种"复杂系统分析工具"。它试图用一套标准化的符号系统来描述和预测人事物的发展变化。无论这种预测的有效性如何，大六壬的符号体系本身就是一种精妙的思维工具，体现了古人对时空、因果、变化的深刻思考。

## 二、大六壬的历史

### 远古起源

大六壬的起源可以追溯到上古时期。据传，六壬源于黄帝时代，是由天神传授给黄帝的占卜之术。当然，这只是传说，但它反映了六壬在传统文化中的崇高地位。

从文献记载来看，大六壬的系统性理论至少在汉代已经形成。东汉的《吴越春秋》中就有关于六壬的记载。到了魏晋南北朝时期，六壬已经成为士大夫阶层常用的占卜方法。

### 唐宋发展

唐宋时期是大六壬发展的黄金时代。这一时期涌现了大量六壬专著，如：

- 《六壬大全》
- 《大六壬指南》
- 《六壬粹言》

唐代的李淳风、袁天罡等著名术数家都精通六壬。宋代的邵康节（邵雍）虽然以梅花易数著称，但也对六壬有深入研究。

### 明清集成

明清时期，大六壬的理论和实践达到了集大成的阶段。这一时期的重要著作包括：

- 《六壬大全》（集历代精华）
- 《壬学琐记》
- 《大六壬探源》

清代学者对六壬进行了系统整理，使其理论更加完善，方法更加规范。

### 现代传承

在现代社会，大六壬因其复杂性而较少被普通人使用，但在术数研究领域仍有重要地位。它的精密逻辑结构吸引了不少研究者，试图从现代视角解读其原理。

## 三、大六壬的核心框架

### 天盘与地盘

大六壬的基础是"天盘"与"地盘"的叠加：

**地盘**：固定不动，按照十二地支顺序排列
**天盘**：根据起课时间旋转，代表当时的天象

天盘的旋转由"月将"确定。月将是太阳所在的位置，随着季节变化而不同。天盘与地盘的叠加，形成了六壬分析的基础框架。

### 十二神将

大六壬使用十二位"神将"来代表不同的力量和因素：

| 神将 | 传统含义 | 现代解读 |
|------|----------|----------|
| 贵人 | 尊贵、帮助 | 贵人相助、上级支持 |
| 螣蛇 | 惊恐、怪异 | 不确定性、意外 |
| 朱雀 | 文书、口舌 | 沟通、信息、争论 |
| 六合 | 和合、婚姻 | 合作、婚恋、交易 |
| 勾陈 | 官讼、战争 | 竞争、冲突、官方 |
| 青龙 | 喜庆、财帛 | 好运、财富、积极 |
| 天空 | 欺诈、虚假 | 空虚、不实、谨慎 |
| 白虎 | 血光、丧亡 | 风险、行动、变动 |
| 太常 | 宴会、衣食 | 生活、享受、稳定 |
| 玄武 | 盗贼、暗昧 | 隐藏、欺骗、暗中 |
| 太阴 | 阴私、谋划 | 策划、女性、隐秘 |
| 天后 | 后宫、女性 | 女性关系、情感 |

这十二神将代表了事物发展中可能涉及的各种力量和因素。

### 四课

四课是大六壬的静态框架，代表事物的四个基本方面：

- **第一课（日干上神）**：代表自己、主动方
- **第二课（日支上神）**：代表他人、被动方
- **第三课（日干三传之初）**：代表起因
- **第四课（日支三传之初）**：代表发展

四课从不同角度描述了问事的基本状况。

### 三传

三传是大六壬的动态核心，代表事物发展的三个阶段：

- **初传**：事情的开始、起因
- **中传**：事情的过程、发展
- **末传**：事情的结果、归宿

三传的推导有九种方法（九种课式），根据四课的阴阳关系来确定。

## 四、大六壬的特点

### 结构精密

大六壬是传统术数中结构最精密的方法之一：

- **多层次**：天盘、地盘、四课、三传、神将等多个层次
- **多维度**：时间、空间、人事、吉凶等多个维度
- **系统性**：各要素之间有严密的逻辑关系

### 信息丰富

一个六壬课包含的信息非常丰富：

- 天盘地盘的叠加关系
- 四课的配置情况
- 三传的发展脉络
- 十二神将的临位
- 各种神煞的影响

这些信息综合起来，可以对问事进行全面的分析。

### 学习难度高

大六壬的学习难度在传统术数中是最高的：

- 需要掌握大量的基础知识
- 需要理解复杂的推导规则
- 需要积累丰富的实践经验

因此，大六壬虽然精密，但传承相对困难。

### 局限性说明

大六壬作为传统预测术，有其固有局限：

1. **复杂性**：复杂不等于准确
2. **主观性**：解读仍有主观空间
3. **不可验证**：许多判断难以客观验证
