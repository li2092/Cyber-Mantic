## 一、起课技巧

### 掌诀记忆法

小六壬的起课需要在手指上点数，熟练掌握手指布局是第一步。

**手指布局**：

```
        食指    中指    无名指
上节    留连    赤口    空亡
下节    大安    速喜    小吉
```

**记忆口诀**：
"大安留连速喜好，赤口小吉空亡了"

起课时从大安开始，按照上述顺序循环点数：
大安→留连→速喜→赤口→小吉→空亡→大安→...

### 快速计算技巧

传统起课需要分三步数月、日、时，但可以用更快的方法：

**直接计算法**：
- 将月+日+时辰数相加
- 用总数除以6，看余数
- 余1=大安，余2=留连，余3=速喜
- 余4=赤口，余5=小吉，余0=空亡

**示例**：农历十一月初六巳时
- 11+6+6=23
- 23÷6=3余5
- 余5=小吉

**注意**：这种方法的结果与传统三步法不同。传统法更重视"层层递进"的过程，直接计算法更快但失去了一些层次感。根据个人习惯选择。

### 农历与公历

小六壬传统上使用农历，因为农历与传统历法系统（天干地支、节气等）一致。

**查找农历的方法**：
- 手机日历大多可以显示农历
- 网上有很多农历公历转换工具
- 黄历App可以直接查看

**如果实在不便查农历**：
- 可以使用公历数字代替（不够传统，但可行）
- 或者使用报数起课法，绕开时间计算

### 时辰判断

时辰是起课的关键要素之一：

**十二时辰对照**：

| 时辰 | 时间段 | 数字 |
|------|--------|------|
| 子时 | 23:00-01:00 | 1 |
| 丑时 | 01:00-03:00 | 2 |
| 寅时 | 03:00-05:00 | 3 |
| 卯时 | 05:00-07:00 | 4 |
| 辰时 | 07:00-09:00 | 5 |
| 巳时 | 09:00-11:00 | 6 |
| 午时 | 11:00-13:00 | 7 |
| 未时 | 13:00-15:00 | 8 |
| 申时 | 15:00-17:00 | 9 |
| 酉时 | 17:00-19:00 | 10 |
| 戌时 | 19:00-21:00 | 11 |
| 亥时 | 21:00-23:00 | 12 |

**记忆技巧**：
- 子时从深夜11点开始
- 每个时辰2小时
- 午时正好是中午11点-下午1点

## 二、灵活起课法

除了时间起课，小六壬还有多种灵活的起课方式：

### 报数起课

最简单的方法——问事者随意报一个数字：

**方法**：
- 问事者心中默想问题
- 脱口报出一个数字
- 用数字除以6，余数定六神

**要点**：
- 数字要脱口而出，不要思考
- 第一个想到的数字最有效
- 可以用任何听到或看到的数字

### 外应起课

根据当下遇到的事物起课：

**可用的外应**：
- 看到的数字（车牌、门牌、价格标签等）
- 听到的声音次数
- 遇到的人数
- 任何可以数的事物

**方法**：
- 问题明确后，关注第一个进入视野的可数事物
- 用该数字起课
- 不要刻意寻找，顺其自然

### 念动起课

心念一动即起课：

**方法**：
- 心中想到问题的那一刻
- 看一下时间
- 以该时间起课

这种方法强调"心动即卦成"，捕捉的是问题浮现的那一刻。

## 三、判断技巧

### 分清问题类型

不同类型的问题，对六神的解读不同：

**急事与缓事**：
- 急事：速喜最佳，留连不利
- 缓事：大安、小吉适宜，速喜未必好

**求进与求稳**：
- 求进取：速喜、小吉为宜
- 求安稳：大安最佳

**主动与被动**：
- 主动出击：忌空亡、留连
- 被动等待：大安、留连可接受

### 综合考量

一次起课的结果不是唯一的判断依据：

**结合实际**：
- 六神只提供倾向，还要看实际情况
- 如果实际条件很差，即使得速喜也要谨慎
- 如果实际条件很好，即使得赤口也不必过虑

**多次参考**：
- 重大事项可以结合其他方法
- 不要只依赖小六壬一种判断
- 小六壬适合快速参考，不适合深度分析

### 时机把握

六神可以提供时机方面的参考：

**行动时机**：
- 得速喜：宜速行动
- 得大安：宜缓行动
- 得留连：宜等待
- 得空亡：宜暂停

**方位参考**：
| 六神 | 参考方位 |
|------|----------|
| 大安 | 东方 |
| 留连 | 东北 |
| 速喜 | 南方 |
| 赤口 | 西方 |
| 小吉 | 北方 |
| 空亡 | 中/西南 |

## 四、常见应用场景

### 日常决策

小六壬最适合日常小事的快速决策：

**今天适合做某事吗？**
- 大安、小吉：适合，平稳
- 速喜：很适合，顺利
- 留连：可以但会慢
- 赤口：小心，可能有摩擦
- 空亡：不太适合

**现在出门好吗？**
- 速喜、小吉：好，顺利
- 大安：好，但不急
- 留连：可以但可能有阻滞
- 赤口：小心，注意交通/口角
- 空亡：不宜

### 选择判断

当面临选择时，可以分别对各选项起课：

**方法**：
- 对选项A起一课
- 对选项B起一课
- 比较两个结果

**注意**：
- 这种方法主观性较强
- 仅供参考，不能作为决定依据
- 如果两个结果都不好，可能都不是好选择

### 寻人寻物

小六壬对寻人寻物有传统的判断方法：

**能否找到**：
- 大安、小吉：能找到
- 速喜：很快找到
- 留连：需要时间
- 赤口：有难度
- 空亡：可能找不到

**方向参考**：
参考六神对应的方位去寻找。

**注意**：
这只是传统说法，实际寻人寻物还是要依靠实际搜寻和报警等正规渠道。

## 五、使用心法

### 诚心为本

虽然小六壬简单，但"诚"依然重要：

- **问题明确**：心中要有具体的问题
- **态度认真**：不是游戏或玩笑
- **不重复起课**：一事一课，不要反复

### 参考而非依赖

小六壬的正确态度是"参考"：

- **不是命令**：结果只供参考，不必完全遵从
- **不要迷信**：它是工具，不是神谕
- **保持理性**：结合实际情况判断
- **不要依赖**：不能事事都问，要培养自主判断力

### 化解与调整

即使结果不佳，也不必恐慌：

**赤口化解**：
- 注意言辞，谨慎沟通
- 提前预防，化解矛盾
- 不是不能做，而是要小心

**空亡化解**：
- 考虑换个时机
- 考虑换个方向
- 有时放下也是智慧

**留连化解**：
- 调整预期，准备等待
- 不急不躁，顺其自然

## 七、常见问题

### 问：结果不好怎么办？

**答**：
1. 结果只是参考，不是定论
2. 不好的结果是提醒，不是诅咒
3. 可以根据提示做出调整
4. 不要被结果困扰，影响心情

### 问：可以重新起课吗？

**答**：
传统上"一事一课"，不宜重复。但如果：
- 情况发生了实质变化
- 已经过了较长时间
- 问的是不同角度的问题

可以重新起课。但不要因为结果不满意就反复起课。

### 问：小六壬准吗？

**答**：
小六壬是传统文化产物，不是科学预测方法。它的"准"不能用科学标准衡量。它的价值在于：
- 提供一个决策参考
- 帮助打破纠结
- 有时能提供有益的提醒

不要过度追求"准不准"，把它当作一个思考工具就好。
