# 大六壬起课方法

> 起课是大六壬分析的第一步——通过精密的计算，将问事时刻的时空信息转化为可分析的符号系统。

---

## 一、起课的基本原理

大六壬的起课是一个将时间信息转化为符号系统的过程。与其他占卜方法不同，大六壬的起课完全基于问事时刻的干支信息，不需要随机过程（如摇卦），因此具有确定性——同一时刻起课，必然得到相同的课式。

起课的核心步骤包括：

1. **确定日干支**：问事当天的天干地支
2. **确定时辰**：问事时刻的地支
3. **确定月将**：当月太阳所在的位置
4. **布置天地盘**：将月将加于时辰之上
5. **立四课**：从日干支推演四课
6. **发三传**：根据四课关系确定三传
7. **配神将**：将十二神将配入天盘

这个过程看似复杂，但每一步都有明确的规则，可以精确计算。

## 二、基础要素

### 日干支

日干支是起课的基础。每一天都有一个对应的天干地支，称为"日柱"。

**十天干**：甲、乙、丙、丁、戊、己、庚、辛、壬、癸
**十二地支**：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥

日干支的查询可以通过万年历或日历App获得。

**日干的作用**：
- 代表问事者自己
- 是推算四课的起点
- 决定神将的起始位置

**日支的作用**：
- 代表问事的对象或环境
- 与日干构成四课的基础

## 三、天地盘的布置

### 地盘

地盘是固定不动的，按照十二地支的顺序排列成环形：

```
        巳  午  未
      辰        申
      卯        酉
      寅        戌
        丑  子  亥
```

地盘代表"地"，是静态的基准。

### 天盘

天盘是旋转的，代表"天"的运动。天盘的布置方法是：

**"月将加时"**：将月将所在的地支对准时辰地支，然后顺时针排列其他地支。

**示例**：
假设月将为午（夏至后），问事时辰为子时。

将"午"对准"子"的位置，然后按顺时针排列：
- 子位上是午
- 丑位上是未
- 寅位上是申
- 以此类推...

天盘布置完成后，就形成了天地盘的叠加状态，这是后续分析的基础。

## 四、四课的建立

四课是从日干支出发，在天地盘上取得的四组关系。

### 四课的结构

| 课位 | 取法 | 含义 |
|------|------|------|
| 第一课 | 日干上神 | 自己、主动 |
| 第二课 | 第一课上神 | 自己的状态延伸 |
| 第三课 | 日支上神 | 他人、被动 |
| 第四课 | 第三课上神 | 他人的状态延伸 |

### 四课的推导

**第一课**：
1. 找日干所寄的地支（日干寄宫）
2. 看该地支上面的天盘地支
3. 这就是"日干上神"

**日干寄宫对照**：

| 日干 | 寄宫 |
|------|------|
| 甲 | 寅 |
| 乙 | 辰 |
| 丙戊 | 巳 |
| 丁己 | 未 |
| 庚 | 申 |
| 辛 | 戌 |
| 壬 | 亥 |
| 癸 | 丑 |

**第二课**：
以第一课得到的地支为基准，看它上面的天盘地支。

**第三课**：
直接看日支上面的天盘地支。

**第四课**：
以第三课得到的地支为基准，看它上面的天盘地支。

### 四课的意义

四课从不同角度描述了问事的基本状况：

- **一、二课**：关于"我"（问事者）的信息
- **三、四课**：关于"彼"（对方/环境）的信息
- **一、三课**：基本状态
- **二、四课**：状态的延伸发展

## 五、三传的发用

三传是大六壬的动态核心，代表事物发展的三个阶段。三传的确定方法称为"发用"，有九种课式。

### 九种课式概述

| 课式 | 条件 | 简述 |
|------|------|------|
| 贼克 | 四课有克 | 取下贼上为初传 |
| 比用 | 有克且多 | 取与日干比者 |
| 涉害 | 比用后仍多 | 取涉害深者 |
| 遥克 | 四课无克 | 取遥克之课 |
| 昴星 | 遥克后仍无 | 取日支上神 |
| 别责 | 特殊情况 | 另有规则 |
| 八专 | 干支同位 | 特殊处理 |
| 返吟 | 天地盘相冲 | 特殊处理 |
| 伏吟 | 天地盘相同 | 特殊处理 |

### 基本原则：贼克

最常用的发用方法是"贼克"——在四课中寻找有克关系的课。

**规则**：
1. 看四课中有无五行相克关系
2. 取"下克上"为初传（称为"贼"）
3. 若无下克上，取"上克下"（称为"克"）
4. 初传确定后，按照一定规则推导中传、末传

### 三传的意义

| 传位 | 名称 | 含义 |
|------|------|------|
| 初传 | 发端 | 事情的起因、开始 |
| 中传 | 移易 | 事情的过程、变化 |
| 末传 | 归计 | 事情的结果、归宿 |

三传描绘了事物从起因到结果的发展过程，是判断事情走向的关键。

1. 根据日干确定贵人的位置

3. 从贵人开始，按顺序配置其他神将
