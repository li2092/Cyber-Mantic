# 测试文档

本目录包含测试相关的文档和模板。

## 文档列表

1. **TESTING_CHECKLIST.md** - 测试检查清单
   - 功能测试检查项
   - 测试前准备清单
   - 测试完成标准

2. **TESTING_REPORT_TEMPLATE.md** - 测试报告模板
   - 标准测试报告格式
   - 填写说明和示例
   - 用于正式测试的文档模板

## 使用说明

### 测试检查清单

在进行功能测试或回归测试前，使用TESTING_CHECKLIST.md确保：
- 所有必要的测试场景已覆盖
- 测试环境已正确配置
- 测试数据已准备

### 测试报告模板

完成测试后，使用TESTING_REPORT_TEMPLATE.md生成正式的测试报告：
1. 复制模板内容
2. 填写测试信息（时间、版本、测试人员）
3. 记录测试结果
4. 列出发现的问题
5. 提供测试结论和建议
